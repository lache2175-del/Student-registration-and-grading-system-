# Mini Project – Student System

## Description
This project is a simple Python-based system designed to manage student registration and grading. It demonstrates basic programming concepts such as data handling, functions, and logic building.

## Objectives
- Practice Python programming
- Understand data management
- Build a simple student system

##  Technologies Used
- Python
- Jupyter Notebook (.ipynb)

##  Project Files
- `Mini Project (2).ipynb` → Main project file
- `mod1 (1).py` → Supporting Python file

##  Features
- Add student information
- Manage student records
- Handle grading system

##  How to Run
1. Open the `.ipynb` file using Jupyter Notebook
2. Run the cells step by step
3. Follow the instructions inside the notebook
